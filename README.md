# Image Uploader

- Vue

## Features

- Загрузка одного или нескольких изображений
- Предпросмотр в модальном окне
- Удаление загруженного изображения
- Изменение порядка изображений

## Installation
```sh
git clone
npm i
npm run serve / npm run build
```
